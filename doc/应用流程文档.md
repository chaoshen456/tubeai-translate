# 应用流程文档

## 新用户引导与注册/登录
新用户通过官方落地页或收藏链接进入系统。落地页简要介绍系统用途，并提供两个明确操作入口：**注册**、**登录**。

用户点击注册后，会展示简易表单，需填写邮箱、密码、确认密码。提交注册后，系统会发送一封邮箱验证链接。用户点击链接激活账号后，会自动重定向到登录页面。

用户也可选择第三方快捷登录，例如 GitHub 授权登录。点击 GitHub 登录按钮、授予权限后，即可完成身份验证并直接进入系统。

在登录页面，可通过账号密码或 GitHub 授权任意一种方式进入系统。
若用户忘记密码，点击「忘记密码」会跳转至邮箱找回表单；输入邮箱后收到重置链接，打开链接设置新密码，即可返回登录页重新登录。

登录后页面顶部会显示**退出登录**按钮，点击可清空登录会话，并跳转回首页落地页。

## 主控仪表盘 / 首页
登录成功后，默认进入系统主控仪表盘。

页面顶部为导航栏：展示系统名称、用户头像菜单，可进入账号设置和退出登录。
左侧是可折叠侧边栏，包含两个导航选项：**仪表盘**、**设置**。

页面中间主体区域以表格形式展示**每日已抓取的视频列表**，每条条目包含：视频缩略图、标题、抓取时间、审核状态。
列表上方提供搜索框和状态筛选下拉菜单。

点击任意视频列表条目，进入视频详情页。
侧边栏始终保持可见，用户可随时在仪表盘和设置页面之间切换。

## 详细功能流程与页面跳转
系统核心由**每日定时无服务器任务**驱动，每天自动运行一次。
该定时任务部署在 Vercel 定时任务 或 Supabase 边缘定时函数上。

任务触发后，调用 YouTube 数据接口，拉取指定地区热门 AI 科技类视频。
对每个视频，优先调用 YouTube 字幕接口自动获取字幕文稿；若无官方字幕，则自动调用 OpenAI Whisper 进行批量语音转文字生成文稿。

完整文稿会按段落拆分，逐段调用 OpenAI GPT-4 翻译接口，生成中文译文。
视频元数据（原文文稿、中文译文、视频标题、YouTube 编号、缩略图地址、抓取时间、审核状态）统一存入 Supabase 的 `videos` 数据表，初始状态标记为**待审核**。

用户从仪表盘点击进入单个视频详情页后，页面分为**左右双栏面板**：
- 左侧：展示带时间戳的原始英文字幕
- 右侧：展示 AI 生成的中文译文，支持在线编辑

面板下方有三个操作按钮：**保存草稿**、**审核通过**、**驳回**。
- 保存草稿：保存当前编辑的译文内容，不改变审核状态
- 审核通过：标记状态为**待发布**，并记录审核时间戳
- 驳回：标记状态为**已驳回**，可填写驳回备注并同步存入数据库

审核通过后，页面会额外展示两个功能按钮：
**生成 Markdown**、**发布到知乎**。
- 生成 Markdown：自动组装带头部元信息、时间戳、译文内容的标准 `.md` 文档
- 发布到知乎：调用 Puppeteer 自动化脚本，登录绑定的知乎账号，自动发布文章内容

页面之间跳转体验无缝流畅。
用户可使用浏览器返回按钮，或点击侧边栏仪表盘链接回到视频列表。
若点击设置页面，可临时离开审核流程，随时切回仪表盘继续审核工作。

## 系统设置与账号管理
在设置页面，用户可编辑个人资料：展示昵称、绑定邮箱。
密码模块支持输入旧密码、设置新密码完成改密。

通知设置提供开关选项，可选择是否接收流水线执行成功/失败的邮件提醒。
所有修改会同步保存到 Supabase 认证信息与 `user_preferences` 用户偏好表。

设置页面顶部提供返回仪表盘的快捷入口。
当前版本暂不包含付费订阅模块，设置均为个人偏好配置，不影响核心视频自动化处理流程。

## 异常场景与容错处理
- 注册或登录时填写非法格式邮箱、错误密码，输入框下方会实时展示行内错误提示，说明具体原因。
- 若定时任务因接口配额超限、网络异常等原因调用 YouTube 或 OpenAI 服务失败：系统自动记录错误日志，采用指数退避策略最多重试 3 次；全部重试失败后，自动向管理员发送邮件告警。
- 审核页面编辑译文、修改审核状态时，若瞬时网络中断操作失败，会弹出全局消息提示，告知操作失败并提示手动重试。
- 无管理员权限的用户若强行访问审核详情地址，会自动重定向至 **403 无权访问** 页面，并提供返回仪表盘的链接。

## 整体流程总结
用户完整使用路径：
注册/登录 → 进入仪表盘查看每日 AI 视频列表 → 进入视频详情查看原文与译文 → 在线编辑译文、保存草稿 → 审核通过/驳回。

审核通过后，可一键生成 Markdown 文档，或直接自动发布到知乎。
用户可随时通过侧边栏进入设置，修改个人资料与消息通知偏好。

后端每日自动化流水线自动完成：视频抓取、语音转文字、AI 翻译、数据入库；人工仅专注**内容审核与发布**。
整体目标：快速完成译文审核并批量发布，让中文用户每日可获取最新 AI 科技资讯内容。
